Release Information
The following features were added in MAGIC Tool 2.0:

  • Gridding has been redesigned.
  • Automatic flagging of spots based on criteria has been implemented at the segmentation stage.
  • In a table, you can search for a gene to have it automatically selected.
  • An option to show gene info in the table generated by the Exploring window has been added.
  • In the View / Edit Data window, an option has been added to allow the highlighting of the top and bottom n genes.
  • You can use the gene alias (if present) in place of the gene name.
  • From the Exploring window, you can save a group as a new expression file.
